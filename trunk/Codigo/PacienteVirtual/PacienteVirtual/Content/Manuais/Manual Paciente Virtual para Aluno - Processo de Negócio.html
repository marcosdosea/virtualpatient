Processo de Negcio
O aluno deve registrar-se no sistema atravs do endereo
(www.pv.sistemasitatechjr.com.br), em seguida solicitar
matrcula na turma desejada.
O prximo passo esperar a matrcula seja aprovada por um
dos tutores da turma solicitada.
Quando o tutor ativar a matrcula do aluno, ele deve atribuir
uma consulta de um paciente para o aluno.
Para visualizar uma consulta atribuda ao aluno, este deve
acessar o sistema fazer o login e acessar o Menu >
Consultas > Consulta.
Processo de Negcio
O aluno deve preencher uma consulta na lista de consultas
destinadas a ele.
Quando o aluno terminar o preenchimento desta consulta,
ele deve encerrar a consulta. Esta consulta ser enviada para a
correo de um tutor.
Caso o tutor tenha observaes, ele poder enviar a consulta
de volta para o aluno, a fim de que este repare sua consulta.
Porm, o tutor tambm poder finalizar esta consulta
encerrando o ciclo da consulta.
