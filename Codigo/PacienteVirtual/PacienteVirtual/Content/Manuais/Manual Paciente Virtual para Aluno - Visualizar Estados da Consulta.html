Manual Paciente Virtual para Aluno
Como visualizar os estados das
Consultas
Na tela de consulta, seguindo pelo menu Consultas >
Consulta. possvel visualizar os estados das consulta.
Nas consultas so encontrados os seguintes estados:
Aguardando o Preenchimento: Quando o tutor atribuiu uma consulta
para o aluno, porm, o aluno ainda no abriu esta consulta para
preencher.
Em Preenchimento: Quando o tutor atribuiu uma consulta para o
aluno, e o mesmo j abriu esta consulta para preencher.
Em Correo: pelo Tutor: Quando o aluno terminou de preencher a
consulta e enviou para a correo do Tutor.
Aguardando Correo do Aluno: Quando o tutor j realizou as
correes necessrias e j enviou de volta para o aluno para a correo
deste.
Finalizado: Quando o tutor e aluno j realizaram todas as correes
necessrias, o tutor finaliza a consulta encerrando o ciclo da consulta.
